5 Abstrak Buku ini mengupas tentang class-class CSS bootstrap dan cara penggunaannya. mulai dari cara mendownload bootstrap, mengintegrasikan dengan jQuery, mendesain web responsive dengan grid system bootstrap, penggunaan class bootstrap untuk merancang navigation, dropdown menu, carousel, modal, panel, icon, form, dan lain sebagainya. buku ini dapat digunakan mahasiswa yang sedang belajar pemrogramanw web dasar (HTML)dan pemrograman web lanjutan (PHP). buku ini ditulis secara sistematis step by step disertai contoh penerapan class-class bootstrap sehinga akan lebih mudah dipahami bagi para pemula dibidang web developer.
